## 一、背景与目标

### 1.1 业务背景

智造云生产运营管理平台的「生产概览页」当前以 KPI 数字卡片和固定表格展示生产数据。随着业务发展，不同角色的管理人员对数据可视化有差异化需求：

- 生产主管关注产线产量对比和完成率趋势
- 设备管理员关注各设备 OEE 排行
- 质量工程师关注不良类型分布和趋势

当前的固定图表无法满足个性化需求，用户需要联系研发人员修改代码才能调整看板内容。

### 1.2 产品目标

为生产概览页增加**自助式可视化仪表盘**能力，让用户无需研发介入即可：

1. 从已有数据源中选择字段，自行搭建图表指标
2. 自由拖拽编排看板布局
3. 持久化保存个人看板配置

### 1.3 衡量指标

| 指标 | 目标 |
|------|------|
| 图表搭建完成率 | 用户从打开搭建器到成功添加图表 ≤ 30 秒 |
| 看板自定义率 | 首周有 > 50% 的活跃用户自定义过看板 |

### 7.1 图表卡片配置（CardConfig）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 唯一标识，新建时用 `Date.now().toString()` |
| title | string | 是 | 图表标题 |

| chartType | enum | 是 | 'bar' \| 'line' \| 'pie' |
| groupByField | string | 是 | X 轴分组维度字段 key |
| valueFields | string[] | 是 | Y 轴数值字段 key 数组 |
| aggregation | enum | 是 | 'sum' \| 'avg' \| 'count' \| 'max' \| 'none' |

### 7.2 看板持久化结构（localStorage）

```json
{
  "version": 2,
  "cards": [
    {
      "config": {
        "id": "default-1",
        "title": "产线产量完成情况",
        "dataSourceId": "line-production",
        "chartType": "bar",
        "groupByField": "lineName",
        "valueFields": ["planned", "actual"],
        "aggregation": "none"
      },
      "grid": { "x": 0, "y": 0, "w": 6, "h": 4 }
    }
  ]
}
```

**键名：** `dashboard-v2`

**容错规则：**
- localStorage 不存在 → 返回默认 3 个卡片
- JSON 解析失败 → 返回默认
- version 不等于 2 → 返回默认
- cards 不是数组 → 返回默认

## 八、聚合计算规则

本期聚合在前端完成（纯函数），后续可迁移到后端。

### 8.1 聚合逻辑

输入：原始数据数组 + groupByField + valueFields + aggregation

处理步骤：
1. 按 `groupByField` 的值将数据分组
2. 对每组中的每个 `valueField` 执行聚合运算
3. 输出聚合后的数组

### 8.2 聚合方式

| 聚合方式 | 说明 | 示例 |
|---------|------|------|
| none | 不聚合，直接透传（仅保留 groupByField + valueFields） | 5 条产线数据原样输出 |
| sum | 分组求和 | 按不良类型汇总不良数量 |
| avg | 分组算术平均 | 按产线求 OEE 平均值 |
| count | 分组计数 | 按检验员统计记录数 |
| max | 分组取最大值 | 按产线取最高产量 |

### 8.3 示例

**输入：** 设备数据，groupByField='lineName'，valueFields=['oee']，aggregation='avg'

**输出：**
```json
[
  { "lineName": "A线-冲压", "oee": 78.6 },
  { "lineName": "B线-焊接", "oee": 69.0 },
  { "lineName": "C线-涂装", "oee": 86.6 },
  { "lineName": "D线-总装", "oee": 73.3 },
  { "lineName": "E线-包装", "oee": 65.4 }
]
```
